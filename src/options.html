Badge Color Unfortuantely, the browser predetermines thefont size of the badge. Setting the badge text color is available in Firefox 63 and above.
Automatically set Badge Color When checked, the text color is either white or black based on the highest constrast with the badge color. When unchecked, a manual color may be selected.
Badge Text Color (if not automatic) This option is ignored/disabled unless "Automatically set Badge Color" is unchecked.

Counter A maximum of four characters can be displayed on the badge; this is a browser limitation.
Exclude Pinned Tabs When checked, pinned tabs are not counted in the badge icon. They will still be counted in the popup.
